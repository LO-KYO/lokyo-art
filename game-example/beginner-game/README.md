# Beginner Game Project

This is a simple beginner-level game project created in Python. The game is designed to help new developers learn the basics of game development using Python.

## Project Structure

```
beginner-game
├── src
│   ├── main.py
│   └── utils.py
├── requirements.txt
└── README.md
```

## Installation

To get started with the game, you need to have Python installed on your machine. You can download it from [python.org](https://www.python.org/downloads/).

### Dependencies

This project requires the following Python libraries:

- pygame

You can install the required dependencies using pip. Run the following command in your terminal:

```
pip install -r requirements.txt
```

## Running the Game

To run the game, navigate to the `src` directory and execute the `main.py` file:

```
cd src
python main.py
```

## Game Instructions

- Use the arrow keys to move your character.
- Collect items to score points.
- Avoid obstacles to stay in the game.
